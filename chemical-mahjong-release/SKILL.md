# 化学麻将 Skill

## 概述
化学元素主题的麻将游戏，用于学习化学元素和化学反应。通过 AI 辅助玩家出牌、吃碰、和牌。

## 游戏规则

### 牌组构成（144张）
| 类型 | 牌 | 数量 |
|------|-----|------|
| 元素牌 | Na, Mg, Al, K, Ca, Fe, Cu, Zn, Ba, Ag, H, P, S, Cl, C, N, O | 各4张（68张） |
| 离子牌 | NO3⁻¹, OH⁻¹, NH4⁺¹, PO4⁻³, CO3⁻², SO4⁻² | 各4张（24张） |
| 分子牌 | KMnO4, K2MnO4, MnO2, CO2, H2O, H2, O2, Cl2 | 各4张（32张） |
| 条件牌 | 点燃, 加热, 高温, 电解 | 各4张（16张） |
| 状态牌 | 气体, 沉淀 | 各2张（4张） |

### 游戏流程
1. **发牌**：庄家14张，闲家各13张
2. **出牌**：逆时针轮流，摸1张出1张
3. **吃碰**：
   - 吃：只能吃上家的牌，组成顺子（3张连续元素）
   - 碰：任意家的牌都可以碰，组成刻子（3张相同）
4. **和牌**：化合价总和为0，总牌数14张

### 基本牌型
- **顺子**：元素周期表上连续的3个元素（如 H-C-N, Na-Mg-Al）
- **刻子**：3张相同的牌
- **对子**：化合价数值相等、符号相反，且组成的物质现实存在

### 和牌类型
| 类型 | 规则 |
|------|------|
| 平和 | 化合价总和为0 |
| 对对胡 | 7组对子 |
| 清一色 | 只有一种牌型（不可吃碰） |
| 一条龙 | 完整化学方程式 + 分子牌 |
| 十三幺 | 特定12元素 + 1个顺子 |

## 重要设定

### ⚠️ 设定1：游戏状态延续性
- **游戏状态必须延续**，不能每一轮重新生成
- 使用固定随机种子或保存游戏状态（pickle）
- 确保玩家的手牌在整个游戏过程中保持一致
- 代码位置：`~/.qclaw/workspace/chemical_mahjong/game_state.pkl`

### ⚠️ 出牌规则（重要！）
- **每次只能出一张牌**，不能出多张
- 即使手牌中有2张相同的牌（如2张Al），出牌时只移除**一张**
- 碰牌时才需要移除2张相同的牌
- 代码示例：
```python
# 错误：移除所有同名牌
hand = [c for c in hand if str(c) != "Al"]  # ❌ 会移除所有Al

# 正确：只移除一张
new_hand = []
removed = False
for c in hand:
    if str(c) == "Al" and not removed:
        removed = True
        continue
    new_hand.append(c)
hand = new_hand  # ✅ 只移除一张Al
```

### ⚠️ 设定2：和牌判定规则
1. **自摸和牌**：摸牌后手牌化合价总和为 0，**且条件牌/状态牌必须全部用于组成化学方程式**
2. **点炮和牌**：其他玩家打出的牌加入手牌后满足上述条件，可和牌
3. **和牌判定时机**：必须在玩家摸牌后判定，不是出牌后

**重要和牌条件：**
- 如果没有条件牌/状态牌 → 只需化合价为0
- 如果有条件牌/状态牌 → 必须同时满足：
  1. 化合价为0
  2. 每张条件牌/状态牌都能与手牌组成化学方程式
  3. 如果某张条件牌/状态牌无法组成化学方程式，则是废牌，必须打掉

**反应条件牌和状态牌的本质：**
- 这些牌只有组成化学方程式才有价值
- 无法组成方程式时就是废牌，必须优先打出
- 出牌优先级：条件牌/状态牌（废牌）> 化合价调整牌 > 其他

### 点炮和牌流程
1. **触发时机**：其他玩家打出一张牌后
2. **触发条件**：
   - 手牌（13张）+ 打出的牌（1张）= 14张
   - 化合价总和为0
   - 条件牌/状态牌全部能组成化学方程式
3. **操作流程**：
   - 从弃牌区拿走这张牌加入手牌
   - 宣布和牌
4. **点炮格式**：
```
【点炮和牌！】

玩家X打出：Y
你手牌（13张）+ Y = 14张
化合价总和：0
条件牌/状态牌全部有用

🎉【点炮和牌！】🎉
🏆 恭喜！你赢了！
```

### ⚠️ 设定2.1：点炮和牌规则
**定义：** 当其他玩家打出一张牌后，如果这张牌加入你的手牌后可以和牌，你可以宣布"和牌"并拿走这张牌。

**触发条件：**
1. 其他玩家刚打出一张牌（在弃牌区）
2. 你当前手牌为13张
3. 将打出的牌加入你的手牌后，手牌变成14张
4. 化合价总和为0
5. 没有无法使用的条件牌/状态牌

**点炮流程：**
```
【第N轮】
玩家X打出：Y

【点炮判定】
你手牌（13张）+ Y = 14张
化合价总和：0 ✅
条件牌/状态牌：无废牌 ✅

【点炮和牌！】
你从弃牌区拿走 Y，宣布和牌！
```

**点炮示例：**
```
你的手牌（13张）：H(+1), Cl(-1), Na(+1), NO3⁻¹(-1), ...
化合价总和：0

玩家2打出：Cl(-1)

【点炮判定】
加入Cl后：化合价总和 = 0 - 1 = -1 ❌ 不能点炮

---

你的手牌（13张）：H(+1), Na(+1), OH⁻¹(-1), NO3⁻¹(-1), ...
化合价总和：1

玩家2打出：Cl(-1)

【点炮判定】
加入Cl后：化合价总和 = 1 - 1 = 0 ✅
没有条件牌/状态牌 ✅
可以点炮！
```

**点炮与碰牌的区别：**
- 碰牌：拿走弃牌，放入吃碰区，然后打出一张牌
- 点炮：拿走弃牌，加入手牌，直接和牌，不再出牌

### ⚠️ 设定3：出牌建议（核心训练目标）
**出牌优先级顺序：**

### 1. 反应条件牌和状态牌（最高优先级）
**核心原则：这些牌只有组成化学方程式才有价值，否则是废牌**

**判断逻辑：**
- 检查手牌中是否有能组成化学方程式的牌组
- 能组成方程式：保留条件牌/状态牌
- 无法组成方程式：**立即打出**（废牌）

**判断方法：**
```
对于每张条件牌/状态牌：
  检查手牌中是否有对应的反应物
  如果有 → 有用牌，保留
  如果没有 → 废牌，立即打出
```

### 2. 化合价调整牌
- 根据当前化合价正负，打出对应符号的牌
- 优先打出化合价绝对值大的牌

### 3. 其他牌
- 保留能组成顺子、刻子、对子的牌
- 保留能组成化学方程式的牌

### 化学方程式简化判断
```
方程式组成要素：
- 点燃 → 需要 H2 + O2
- 加热/高温 → 需要 KMnO4 或其他可加热物质
- 电解 → 需要水或可电解物质
- 气体 → 需要产生气体的反应
- 沉淀 → 需要产生沉淀的反应
```

### 出牌建议示例
```
手牌：H2, O2, 点燃, Na, Cl
分析：
  - 点燃：有H2+O2 → 保留
  - 建议出 Na 或 Cl（化合价调整）

手牌：Na, Cl, 点燃, Ca
分析：
  - 点燃：没有H2/O2/KMnO4等 → 废牌！
  - 建议出 点燃
```

### ⚠️ 设定4：AI 吃碰逻辑
- **碰**：任意玩家打出的牌，手中有2张相同即可碰
- **吃**：只能吃上家的牌，组成顺子（3张连续元素）
- **AI 吃碰判定**：AI 玩家根据手牌判断是否吃碰
- **吃碰区**：吃碰后的3张牌组放入 `eat_peng_area`，不在手牌中
- **和牌时**：手牌 + 吃碰区 = 14张，化合价总和为 0

### ⚠️ 设定5：模拟真实性要求
**牌局模拟必须具有挑战性，不能太简单：**
1. **初始手牌随机生成**：确保每次游戏不同
2. **AI 玩家有真实决策**：AI 根据化合价和吃碰判断是否出牌
3. **和牌难度适中**：
   - 不是每次都能轻易和牌
   - 需要玩家做出正确的出牌选择
   - AI 玩家也会尝试和牌
4. **训练目的**：
   - 训练给出准确出牌建议
   - 为树莓派硬件实施做准备
   - 模拟真实对战场景

## ⚠️ 设定7：吃碰牌逻辑（固化）

### 正常摸牌出牌流程
```
手牌13张 → 摸牌 → 手牌14张 → 出牌 → 手牌13张
```

### 碰牌规则
1. **触发时机**：其他玩家打出一张牌后，**我摸牌前**
2. **触发条件**：我手牌（13张）中有2张与打出的牌相同的牌
3. **操作流程**：
   - 从手牌（13张）中移除2张相同的牌 → 手牌变成11张
   - 加上其他玩家打出的1张牌，共3张放入吃碰牌区
   - **直接打出一张手牌**（本轮不摸牌）→ 手牌变成10张
   - 下一轮正常摸牌 → 手牌11张 → 出牌 → 手牌10张
4. **碰牌格式**：
```
【碰牌操作】
你手牌中有2张X，碰玩家Y打出的X，共3张X放入吃碰牌区

【你的手牌】（11张）
| 类型 | 牌 |
|------|-----|
| **元素** | a, b, c |
...

总牌数：11张，化合价总和：±N

【你的吃碰牌区】
  第1组：X X X（碰牌）

【出牌建议】
出 Y（+N）

请告诉我你要出哪张牌（碰牌后必须直接出牌，本轮不摸牌）
```

### 吃牌规则
1. **触发时机**：**上家**打出一张牌后，**我摸牌前**
2. **触发条件**：
   - 只能吃**上家**打出的牌
   - 我手牌（13张）中有2张牌能与打出的牌组成顺子（3张连续元素）
   - 顺子定义：元素周期表上连续的3个元素（如 Na-Mg-Al, H-He-Li 不对，应该是按原子序数连续）
   - 实际顺子例子：Na(11)-Mg(12)-Al(13), H(1)-He(2)-Li(3), C(6)-N(7)-O(8), S(16)-Cl(17)-Ar(18无牌)
   - 当前牌库元素：Na(11), Mg(12), Al(13), K(19), Ca(20), Fe(26), Cu(29), Zn(30), Ba(56), Ag(47), H(1), P(15), S(16), Cl(17), C(6), N(7), O(8)
   - 可组成顺子的元素组：Na-Mg-Al, H-C-N, C-N-O, P-S-Cl
3. **操作流程**：
   - 从手牌（13张）中移除组成顺子的2张牌 → 手牌变成11张
   - 加上上家打出的1张牌，共3张放入吃碰牌区
   - **直接打出一张手牌**（本轮不摸牌）→ 手牌变成10张
4. **吃牌示例**：
```
上家打出：Mg
你手牌中有：Na, Al
可以吃牌！Na-Mg-Al 组成顺子

【吃牌操作】
从手牌移除 Na 和 Al，加上上家打出的 Mg，共3张放入吃碰牌区

【你的手牌】（11张）
...

【你的吃碰牌区】
  第1组：Na Mg Al（吃牌，顺子）

【出牌建议】
出 X（+N）

请告诉我你要出哪张牌（吃牌后必须直接出牌，本轮不摸牌）
```

你选择：碰加热

【你的吃碰牌区】
  第1组：加热 加热 加热（碰牌）

### 吃碰后的手牌数量变化（重要！）

**正常情况（无吃碰）：**
```
初始（庄家）：手牌14张 → 出牌 → 手牌13张
每一轮：手牌13张 → 摸牌14张 → 出牌13张 → 手牌13张（循环）
```

**碰/吃牌后：**
```
触发时机：其他玩家打出牌后，我摸牌前
手牌13张 → 碰/吃（移除2张）→ 手牌11张 → 直接出牌 → 手牌10张
下一轮：手牌10张 → 摸牌11张 → 出牌10张 → 手牌10张（循环）
```

**关键点：**
- 碰/吃牌发生在**我摸牌前**，所以本轮不再摸牌
- 碰/吃后必须**立即出牌**，不能跳过
- 碰/吃后手牌数量减少3张（移除2张 + 出牌1张）

### 吃碰后的和牌判定
- 和牌时：手牌 + 吃碰牌区 = 14张
- 化合价总和为0（包括手牌和吃碰牌区的化合价）
- 例如：吃碰区3张 + 手牌11张 = 14张

### 点炮和牌判定
- **触发时机**：其他玩家打出一张牌后，我摸牌前
- **触发条件**：
  - 手牌 + 吃碰区 + 打出的牌 = 14张
  - 化合价总和为0（手牌 + 吃碰区 + 打出的牌）
  - 条件牌/状态牌全部能组成化学方程式
- **操作流程**：
  - 从弃牌区拿走这张牌加入手牌
  - 宣布和牌
- **示例**：
```
【点炮和牌！】

玩家X打出：Y
手牌（11张）+ 吃碰区（3张）+ Y（1张）= 15张 ❌ 错误！

正确：
手牌（10张）+ 吃碰区（3张）+ Y（1张）= 14张 ✅
化合价总和：0
条件牌/状态牌全部有用

🎉【点炮和牌！】🎉
```
