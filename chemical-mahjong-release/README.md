# 化学麻将 (Chemical Mahjong) 🧪🀄

将麻将规则与化学知识结合的创意教育游戏！

## 游戏简介

化学麻将是一款结合中国麻将和化学元素知识的创新游戏。玩家需要通过出牌、吃碰，最终让手牌的化合价总和为0来实现和牌。

## 游戏特色

- 🧪 **化学元素学习**：熟悉元素周期表和化合价
- 🎴 **麻将策略**：传统麻将的吃碰规则
- 🎯 **化学方程式**：组合出真实的化学反应
- 🤖 **AI辅助**：智能推荐出牌策略

## 牌组构成（144张）

| 类型 | 牌 | 数量 |
|------|-----|------|
| 元素牌 | Na, Mg, Al, K, Ca, Fe, Cu, Zn, Ba, Ag, H, P, S, Cl, C, N, O | 各4张（68张） |
| 离子牌 | NO3⁻¹, OH⁻¹, NH4⁺¹, PO4⁻³, CO3⁻², SO4⁻² | 各4张（24张） |
| 分子牌 | KMnO4, K2MnO4, MnO2, CO2, H2O, H2, O2, Cl2 | 各4张（32张） |
| 条件牌 | 点燃, 加热, 高温, 电解 | 各4张（16张） |
| 状态牌 | 气体, 沉淀 | 各2张（4张） |

## 游戏规则

### 基本流程
1. **发牌**：庄家14张，闲家各13张
2. **出牌**：逆时针轮流，摸1张出1张
3. **吃碰**：
   - **吃牌**：只能吃上家的牌，组成顺子（元素周期表连续3元素）
   - **碰牌**：任意家的牌都可以碰，组成刻子（3张相同）
4. **和牌**：化合价总和为0，总牌数14张

### 和牌条件
- 手牌化合价总和为0
- 条件牌/状态牌必须能组成有效化学方程式
- 总牌数为14张

### 和牌类型
| 类型 | 规则 | 番数 |
|------|------|------|
| 平和 | 化合价总和为0 | 1番 |
| 对对胡 | 7组对子 | 2番 |
| 清一色 | 只有一种牌型（不可吃碰） | 3番 |
| 一条龙 | 完整化学方程式 + 分子牌 | 4番 |

## 安装方法

### 方法一：从本地安装
```bash
# 将此目录复制到 OpenClaw 的 skills 目录
cp -r chemical-mahjong ~/.qclaw/workspace/skills/
```

### 方法二：从 GitHub 安装
```bash
openclaw skills install https://github.com/Willyco/chemical-mahjong.git
```

## 使用方法

在 OpenClaw 中说：
- "开始化学麻将" - 开始新游戏
- "出 Na" - 打出一张牌
- "碰" / "吃" - 进行吃碰操作
- "和牌" - 宣布和牌
