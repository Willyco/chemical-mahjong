# 化学麻将 (Chemical Mahjong) 🧪🀄

将麻将规则与化学知识结合的创意教育游戏！

## 游戏简介

化学麻将是一款结合中国麻将和化学元素知识的创新游戏。玩家需要通过出牌、吃碰，最终让手牌的化合价总和为0来实现和牌。

## 游戏特色

- 🧪 **化学元素学习**：熟悉元素周期表和化合价
- 🎴 **麻将策略**：传统麻将的吃碰规则
- 🎯 **化学方程式**：组合出真实的化学反应
- 🤖 **AI辅助**：智能推荐出牌策略

## 安装方法

```bash
openclaw skills install https://github.com/Willyco/chemical-mahjong.git
```

## 使用方法

在 OpenClaw 中说：
- "开始化学麻将" - 开始新游戏
- "出 Na" - 打出一张牌
- "碰" / "吃" - 进行吃碰操作
- "和牌" - 宣布和牌
